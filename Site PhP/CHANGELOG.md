### 🎨 Sistema de Cores

#### Variáveis CSS Implementadas
```css

--foreground: Cor do texto principal

--card-foreground: Cor do texto dos cards
--primary: Cor primária da marca
--secondary: Cor secundária
--muted: Cores esmaecidas para elementos secundários
--border: Cor das bordas
--input: Cor de fundo dos inputs
--ring: Cor do foco dos elementos
```
